# Git

<hr>

## Git 이란
- 리누스 토발즈가 개발한 **분산형 버전 관리 시스템(DVCS)** 으로, *코드의 변경 사항을 추적하고 여러 사람이 동시에 협업* 할 수 있게 돕는 도구이다.
- 파일의 수정, 추가, 삭제 기록을 관리하며, 로컬 환경에서 신속하게 작업하고 원격 저장소와 동기화하여 프로젝트의 안정적인 운영을 지원한다.

<hr>

## Git의 기본 명령어
- **_git init_** : 현재 디렉토리를 **git 저장소로 초기화**한다.
- **_git clone_** <원격 레포지토리 주소> : 원격 저장소를 **로컬로 복제**한다.
- **_git status_** : 현재 **브랜치의 상태를 확인**한다.
- **_git add_** <파일명> : **특정 파일**을 스테이징 영역에 추가한다.
- **_git add_** : **변경된 모든 파일**을 스테이징 영역에 추가한다.
- **_git commit -m "커밋메시지"_** : **스테이징 변경 사항**을 해당 커밋메시지로 커밋한다.
- **_git log_** : **커밋 기록**을 확인한다.
- **_git show <커밋ID>_** : **특정 커밋의 상제 정보**를 확인한다.
