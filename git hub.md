# Git Hub

<hr>

## Git Hub란

- 분산 버전 관리 시스템인 **Git(깃)을 사용하는 프로젝트를 저장**하고, 전 세계 개발자들과 협업 및 소스 코드를 공유할 수 있는 **웹 기반 호스팅 플랫폼**이다.

<hr>

## Git Hub의 주요 특징

- Git 원격 저장소 : 프로젝트 코드의 버전 변경 사항을 온**라인(클라우드)에 저장하고 관리**한다.
- 협업 기능 : 팀원들과 **코드를 병합하고, 리뷰 및 논의를 할 수 있는 플랫폼**을 제공한다.
- 오픈 소스 및 소셜 코딩 : 수많은 공개 **프로젝트를 검색하고 기여**할 수 있다.
- 포트폴리오 역할 : 개발자의 코드 기여도와 프로젝트 활동을 기록하여 취업이나 이직시 **역량을 보여주는 용도**로 사용된다.
- 기타 기능 : 이슈 트래킹, 코드 자동화, 위키 및 프로젝트 페이지를 제공한다

<hr>

## Git과 Git Hub의 차이점

- Git (도구)
  - 역할 : 로컬 컴퓨터(내 PC)에서 **소스 코드 변경 사항을 추적 및 관리**한다
  - 방식 : 설치하여 사용하는 **분산 버전 관리 시스템**이다.
  - 특징 : **오프라인 환경에서도 작동**하며, 커밋, 브랜치등의 작업을 수행한다.
- Git Hub (서비스)
  - 역할 : Git 저장소를 클라우드에 호스팅하여 **다른 사람과 협업 및 공유** 할 수 있다.
  - 방식 : **웹 기반의 저장소 호스팅 서비스**이다.
  - 특징 : 온라인 저장소 제공, Pull Request 코드 리뷰, Issue 등 협업 도구를 제공한다.
